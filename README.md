# WordPress Useful Snippets

私がWordPress開発で頻繁に利用する、保守性とパフォーマンスを重視したコードスニペット集です。

## 収録内容
- **enqueue-scripts.php**: CSSやJavaScriptを安全に読み込むための定義ファイルです。

## こだわりポイント
- **パフォーマンスの最適化**: `is_home()` や `is_front_page()` などの条件分岐を活用し、必要なページでのみリソースを読み込むことで、サイト全体の表示速度低下を防ぐ実装をしています。
- **安全性**: 外部スクリプトの管理や、依存関係の制御を徹底しています。
